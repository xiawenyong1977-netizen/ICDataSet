# 预训练模型信息

## StyleGAN2模型
- ffhq.pkl: StyleGAN2-ADA FFHQ数据集预训练模型
- celebahq.pkl: StyleGAN2-ADA CelebA-HQ数据集预训练模型

## Torch-Hub模型
- progan_celebahq.pth: ProGAN CelebA-HQ 512x512预训练模型

## 使用方法
1. 在face_gen.py中修改模型路径
2. 确保CUDA环境可用
3. 运行程序生成高质量人脸图像

## 模型来源
- StyleGAN2: NVIDIA Research
- ProGAN: Facebook Research
